# NumPy & Pandas & PyTorch 学习项目

本项目包含 NumPy、Pandas 和 PyTorch 的学习代码和示例。

## 📁 项目结构

```
├── numpy_learning/      # NumPy 学习示例
│   ├── numpy_1.py      # 数组基础操作
│   ├── numpy_2.py      # 数组索引和切片
│   ├── numpy_3.py      # 数组运算
│   ├── numpy_4.py      # 矩阵运算
│   ├── numpy_5.py      # 随机数生成
│   ├── numpy_6.py      # 数组操作
│   ├── numpy_7.py      # 线性代数
│   └── numpy_8.py      # 综合应用
│
├── pandas_learning/     # Pandas 学习示例
│   ├── pandas_1.py     # Series 和 DataFrame 基础
│   ├── pandas_2.py     # 数据选择
│   ├── pandas_3.py     # 数据处理
│   ├── pandas_4.py     # 数据清洗
│   ├── pandas_5.py     # 数据合并
│   └── pandas_6.py     # 数据分析
│
└── pytorch_learning/    # PyTorch 学习示例
    ├── 1.py            # 自定义数据集
    ├── 2.py - 15.py    # PyTorch 各种功能示例
    ├── model.py        # 模型定义
    └── model_v2.py     # 模型优化版本
```

## 🚀 快速开始

### 环境要求

- Python 3.8+
- NumPy
- Pandas
- PyTorch
- Pillow

### 安装依赖

```bash
pip install numpy pandas torch pillow
```

## 📚 使用说明

### NumPy 示例

```bash
cd numpy_learning
python numpy_1.py
```

### Pandas 示例

```bash
cd pandas_learning
python pandas_1.py
```

### PyTorch 示例

```bash
cd pytorch_learning
python 1.py
```
